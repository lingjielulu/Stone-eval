# 公共资源

这里管理跨实验使用或由公共评测代码直接依赖的数据。

```text
resources/
├── corpora/            小说等文本语料库
│   └── hongloumeng/    红楼梦语料及派生格式
└── lexicons/
    └── ntusd/          情感分析正负面词表
```

- 大体积原始语料和派生文件仅在本地保留。
- 新增小说或其他长文本语料时，在 `corpora/` 下建立独立目录。
- NTUSD 的 UTF-8 运行版本纳入版本管理。
- 评测代码、参数说明和运行记录保存在对应的 `stone_eval/` 功能模块。
