# Poetry Dataset

`resources/corpora/poetry/` 只保存诗歌类语料。小说语料位于对应伏笔工作区
或其他 `corpora/` 子目录：

- 短篇小说：[`foreshadow/short_stories/dataset/`](../../../foreshadow/short_stories/dataset/)
- 契诃夫：[`foreshadow/chekhov/dataset/`](../../../foreshadow/chekhov/dataset/)
- 红楼梦公共语料：[`resources/corpora/hongloumeng/`](../hongloumeng/)

诗歌目录：

- [`classical/`](classical/)：古典诗词原文、规范化文本与伏笔标注。
- [`hongloumeng/`](hongloumeng/)：《红楼梦》诗词原文、规范化文本与伏笔标注。
